7. Recursos necesarios

Debes conocer

Los recursos que necesita el alumnado son:

  • Ordenador con acceso a Internet.
  • Acceso a la plataforma Moodle.
  • Software referente a seguridad informática que el profesor proporcionará a lo largo del curso.
  • Sistema operativo Windows 7/8/10, Windows 2008/2012 Server y Ubuntu 16.04 Linux.
  • Navegador Internet Explorer/Mozilla/Chrome/Opera/Safari para el acceso a Internet
  • Suite OpenOffice.
  • Herramientas software de seguridad.
  • Programa VirtualBox/VMware para la gestión y configuración de máquinas virtuales.
